# 📝 Words Counter Tool

A lightweight and responsive web-based tool to analyze text. It calculates:

- ✅ Word Count
- ✅ Character Count
- ✅ Sentence Count
- ✅ Paragraph Count
- ✅ Space Count

Perfect for writers, editors, students, and anyone who needs a quick, live analysis of their text.
